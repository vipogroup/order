# VIPO Inventory Manager

אתר HTML סטטי לניהול מלאי ומכירות.

## מה כלול

- `index.html` — עמוד האתר הראשי.
- `css/styles.css` — עיצוב מלא.
- `js/app.js` — לוגיקה: מלאי פתיחה, מכירות, חישוב נותר, יצוא/יבוא.
- `assets/order-father.jpeg` — תמונת המקור של הזמנה אבא.
- `data/opening-inventory.csv` — טבלת מלאי פתיחה בקובץ CSV.

## מלאי פתיחה שהוזן

- הזמנה אבא: 191 יחידות לפי התמונה.
- מלאי דורון: 78 שולחנות דורון במידה 180*90*70.
- סה״כ מלאי פתיחה: 269 יחידות.
- עלות ידועה להזמנה אבא: 14,359.71 דולר. לא כולל דורון כי לא נמסר מחיר עלות.

## העלאה ל-GitHub Pages

1. פתח Repository חדש ב-GitHub.
2. העלה את כל הקבצים והתיקיות מתוך הזיפ לשורש ה-Repository.
3. עבור אל Settings → Pages.
4. בחר Deploy from a branch.
5. בחר Branch: `main`, Folder: `/root`.
6. שמור והמתן לקישור האתר.

## שמירת נתונים

האתר שומר מכירות בדפדפן באמצעות LocalStorage. כדי לא לאבד נתונים:

- לחץ באתר על "יצוא גיבוי JSON" בסוף כל יום עבודה.
- כדי להעביר למחשב אחר, השתמש ב-"יבוא גיבוי".
